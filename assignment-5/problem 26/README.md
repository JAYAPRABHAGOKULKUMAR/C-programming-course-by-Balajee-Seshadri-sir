## Problem Statement
Write a program to print the **biggest 4-digit number divisible by 7 and 9**.

## Explanation
A number divisible by both 7 and 9 must be divisible by 63.

## Approach
1. Start from 9999.
2. Check divisibility by 63.
3. Print the first number that satisfies the condition.
