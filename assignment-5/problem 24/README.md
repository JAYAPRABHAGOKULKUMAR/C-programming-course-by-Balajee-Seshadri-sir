## Problem Statement
Write a program to get a number from the user and print the **total number of two-digit perfect square numbers** in the number.

## Explanation
Two-digit perfect square numbers are:
16, 25, 36, 49, 64, 81

The program extracts digits in pairs and checks whether they form a two-digit perfect square.

## Approach
1. Read the number.
2. Extract digits one by one.
3. Check if the digit is a two-digit perfect square.
4. Count and print the result.
