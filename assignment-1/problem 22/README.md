# C Level 1 – Problem 22

## Problem Statement
Subtract 5 from the number **if the ten’s digit is odd**, then print it.  
Do not use `if`.

## Example

| Input | Output |
|------:|-------:|
| 685   | 685    |
| 89172 | 89167  |

## Approach
1. Extract the ten’s digit.
2. Check if it is odd.
3. Subtract 5 accordingly.

## Concepts Used
- Modulo operator
- Integer division
