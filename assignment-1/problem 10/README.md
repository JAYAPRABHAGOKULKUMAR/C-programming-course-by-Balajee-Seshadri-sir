# C Level 1 – Problem 10

## Problem Statement
Get a three-digit number from the user and print the **ten’s digit**.

## Example

| Input | Output |
|------:|-------:|
| 456   | 5      |
| 569   | 6      |

## Approach
1. Read a three-digit integer.
2. Divide the number by `10`.
3. Use modulo `10` to extract the ten’s digit.
4. Print the result.

## Concepts Used
- Integer division
- Modulo operator
