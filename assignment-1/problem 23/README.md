## Problem Statement
Subtract 5 from a two-digit number **if the sum of its digits is odd**, then print it.  
Do not use `if`.

## Example

| Input | Output |
|------:|-------:|
| 95    | 95     |
| 72    | 67     |

## Approach
1. Extract both digits.
2. Add the digits.
3. Check odd condition.
4. Subtract 5 accordingly.

## Concepts Used
- Modulo operator
- Integer arithmetic
