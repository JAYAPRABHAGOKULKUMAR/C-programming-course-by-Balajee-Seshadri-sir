# C Level 1 – Problem 13

## Problem Statement
Get a two-digit number from the user and print the **reverse of the number**.

## Example

| Input | Output |
|------:|-------:|
| 56    | 65     |
| 59    | 95     |

## Approach
1. Read a two-digit integer.
2. Extract the ten’s and one’s digits.
3. Swap the digits.
4. Print the reversed number.

## Concepts Used
- Integer division
- Modulo operator
