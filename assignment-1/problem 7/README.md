# C Level 1 – Problem 7

## Problem Statement
Get a two-digit number from the user and print the **ten’s digit**.

## Example

| Input | Output |
|------:|-------:|
| 45    | 4      |
| 56    | 5      |

## Approach
1. Read a two-digit integer.
2. Divide the number by `10`.
3. Print the quotient.

## Concepts Used
- Integer division
