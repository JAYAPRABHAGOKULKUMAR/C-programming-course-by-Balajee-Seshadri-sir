## Problem Statement
Get a three-digit number from the user and repeatedly sum its digits until a **single-digit result** is obtained.

## Example

| Input | Output |
|------:|-------:|
| 123   | 6      |
| 149   | 5      |
| 991   | 1      |

## Approach
1. Add the digits of the number.
2. If the sum has more than one digit, add its digits again.
3. Repeat until a single digit is obtained.

## Concepts Used
- Integer division
- Modulo operator
