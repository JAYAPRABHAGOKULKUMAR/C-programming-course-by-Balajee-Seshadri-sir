# C Level 1 – Problem 19

## Problem Statement
Get a three-digit number from the user, make the **one’s digit 2**, and print it.

## Example

| Input | Output |
|------:|-------:|
| 695   | 692    |
| 181   | 182    |

## Approach
1. Read a three-digit number.
2. Remove the one’s digit.
3. Add 2 in the one’s position.
4. Print the result.

## Concepts Used
- Integer division
