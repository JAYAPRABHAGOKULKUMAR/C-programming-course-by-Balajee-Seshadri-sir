# C Level 1 – Problem 17

## Problem Statement
Get a two-digit number from the user, make the **one’s digit 0**, and print it.

## Example

| Input | Output |
|------:|-------:|
| 95    | 90     |
| 18    | 10     |

## Approach
1. Read a two-digit number.
2. Remove the one’s digit.
3. Print the modified number.

## Concepts Used
- Integer division
