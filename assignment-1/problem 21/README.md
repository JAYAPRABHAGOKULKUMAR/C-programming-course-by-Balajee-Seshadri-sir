## Problem Statement
Get a number from the user and subtract 5 **if the number is odd**, then print the result.  
Do not use `if`.

## Example

| Input | Output |
|------:|-------:|
| 695   | 690    |
| 182   | 182    |

## Approach
1. Read a number.
2. Check odd condition using modulo.
3. Multiply result with 5.
4. Subtract from the number.

## Concepts Used
- Modulo operator
- Arithmetic logic
