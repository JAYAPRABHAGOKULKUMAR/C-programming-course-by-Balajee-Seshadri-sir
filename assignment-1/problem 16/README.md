## Problem Statement
Get a four-digit number from the user and reverse **only the last two digits**, then print the number.

## Example

| Input | Output |
|------:|-------:|
| 9561  | 5961   |
| 3859  | 8359   |

## Approach
1. Read a four-digit number.
2. Extract all digits.
3. Reverse only the last two digits.
4. Print the result.

## Concepts Used
- Integer division
- Modulo operator
