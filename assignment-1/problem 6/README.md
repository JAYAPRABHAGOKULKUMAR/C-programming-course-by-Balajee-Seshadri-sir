# C Level 1 – Problem 6

## Problem Statement
Get a two-digit number from the user and print the **one’s digit**.

## Example

| Input | Output |
|------:|-------:|
| 45    | 5      |
| 56    | 6      |

## Approach
1. Read a two-digit integer.
2. Use `% 10` to extract the one’s digit.
3. Print the result.

## Concepts Used
- Modulo operator
