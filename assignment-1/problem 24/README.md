# C Level 1 – Problem 24

## Problem Statement
Subtract 5 from a three-digit number **if both hundred’s and one’s digits are odd**, then print it.  
Do not use `if`.

## Example

| Input | Output |
|------:|-------:|
| 795   | 790    |
| 372   | 372    |

## Approach
1. Extract hundred’s and one’s digits.
2. Check if both are odd.
3. Subtract 5 accordingly.

## Concepts Used
- Modulo operator
- Logical multiplication
