# C Level 1 – Problem 20

## Problem Statement
Get a three-digit number from the user and make the **ten’s digit 0**, then print it.

## Example

| Input | Output |
|------:|-------:|
| 695   | 605    |
| 182   | 102    |

## Approach
1. Read a three-digit number.
2. Extract hundred’s and one’s digits.
3. Set the ten’s digit to 0.
4. Print the result.

## Concepts Used
- Integer division
- Modulo operator
