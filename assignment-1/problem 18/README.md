# C Level 1 – Problem 18

## Problem Statement
Get a two-digit number from the user, make the **ten’s digit 1**, and print it.

## Example

| Input | Output |
|------:|-------:|
| 95    | 15     |
| 82    | 12     |

## Approach
1. Read a two-digit number.
2. Replace the ten’s digit with 1.
3. Print the result.

## Concepts Used
- Modulo operator
