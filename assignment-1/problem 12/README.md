# C Level 1 – Problem 12

## Problem Statement
Get a three-digit number from the user and print the **sum of the digits**.

## Example

| Input | Output |
|------:|-------:|
| 562   | 13     |
| 469   | 19     |

## Approach
1. Read a three-digit integer.
2. Extract the hundred’s, ten’s, and one’s digits.
3. Add all three digits.
4. Print the sum.

## Concepts Used
- Integer division
- Modulo operator
