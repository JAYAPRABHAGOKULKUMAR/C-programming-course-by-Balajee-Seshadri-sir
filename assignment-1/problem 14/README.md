# C Level 1 – Problem 14

## Problem Statement
Get a three-digit number from the user and print the **reverse of the number**.

## Example

| Input | Output |
|------:|-------:|
| 561   | 165    |
| 859   | 958    |

## Approach
1. Read a three-digit integer.
2. Extract the hundred’s, ten’s, and one’s digits.
3. Rearrange the digits in reverse order.
4. Print the reversed number.

## Concepts Used
- Integer division
- Modulo operator
