# C Level 1 – Problem 15

## Problem Statement
Get a four-digit number from the user and **reverse only the first two digits**, then print the number.

## Example

| Input | Output |
|------:|-------:|
| 9561  | 9516   |
| 3859  | 3895   |

## Approach
1. Read a four-digit integer.
2. Extract all four digits.
3. Swap only the thousand’s and hundred’s digits.
4. Keep the last two digits unchanged.
5. Print the result.

## Concepts Used
- Integer division
- Modulo operator
