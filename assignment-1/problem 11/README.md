# C Level 1 – Problem 11

## Problem Statement
Get a three-digit number from the user and print the **one’s digit**.

## Example

| Input | Output |
|------:|-------:|
| 456   | 6      |
| 569   | 9      |

## Approach
1. Read a three-digit integer.
2. Use modulo `10` to extract the one’s digit.
3. Print the result.

## Concepts Used
- Modulo operator
