# C Level 1 – Problem 27

## Problem Statement
Get a two-digit number from the user and print the **sum of the digits**.

## Example

| Input | Output |
|------:|-------:|
| 56    | 11     |
| 69    | 15     |

## Approach
1. Read a two-digit integer.
2. Extract the ten’s digit using integer division.
3. Extract the one’s digit using modulo.
4. Add both digits.
5. Print the sum.

## Concepts Used
- Integer division
- Modulo operator
