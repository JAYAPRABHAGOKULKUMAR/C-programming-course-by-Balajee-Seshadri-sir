# C Level 1 – Problem 9

## Problem Statement
Get a three-digit number from the user and print the **hundred’s digit**.

## Example

| Input | Output |
|------:|-------:|
| 456   | 4      |
| 569   | 5      |

## Approach
1. Read a three-digit integer.
2. Divide the number by `100`.
3. Print the quotient.

## Concepts Used
- Integer division
