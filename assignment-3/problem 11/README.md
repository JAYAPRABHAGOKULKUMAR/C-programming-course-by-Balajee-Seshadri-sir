# C Level 2 – Problem 11

## Problem Statement
Get a two-digit number from the user and check if **digit 1 is less than or equal to digit 0**.  
If yes, print **"Success"**; otherwise, print **"Failure"**.

## Example

| Input | Output   |
|------:|----------|
| 66    | Success  |
| 67    | Success  |
| 98    | Failure  |

## Approach
1. Read the number.
2. Extract digit 0 and digit 1.
3. Compare digit 1 ≤ digit 0.
4. Print the result.

## Concepts Used
- Integer division
- Modulus operator
- if/else
