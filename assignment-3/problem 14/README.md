## Problem Statement
Get a four-digit number from the user and check if the **first two digits and last two digits are the same**.  
If yes, print **"Success"**; otherwise, print **"Failure"**.

## Example

| Input | Output   |
|------:|----------|
| 1919  | Success  |
| 6567  | Failure  |

## Approach
1. Split the number into first two digits and last two digits.
2. Compare both values.
3. Print result.

## Concepts Used
- Integer division
- Modulus
- if/else
