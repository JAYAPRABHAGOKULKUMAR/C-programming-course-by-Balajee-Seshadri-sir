# C Level 2 – Problem 12

## Problem Statement
Get a two-digit number from the user and check if **digit 1 is greater than or equal to digit 0**.  
If yes, print **"Success"**; otherwise, print **"Failure"**.

## Example

| Input | Output   |
|------:|----------|
| 66    | Success  |
| 67    | Failure  |
| 98    | Success  |

## Approach
1. Extract digits.
2. Compare digit 1 ≥ digit 0.
3. Print result.

## Concepts Used
- Arithmetic operators
- Conditional statements
