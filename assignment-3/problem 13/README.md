## Problem Statement
Get a four-digit number from the user and check if **digit 1 and digit 0 are the same and also different**.  
If yes, print **"Success"**; otherwise, print **"Failure"**.

## Example

| Input | Output   |
|------:|----------|
| 988   | Success  |
| 567   | Failure  |

## Approach
1. Extract digit 0 and digit 1.
2. Check equality and inequality conditions.
3. Print result.

## Concepts Used
- Modulus
- Integer division
- Logical operators
