## Problem Statement
Get a two-digit number from the user and check if **digit 0 is greater than digit 1**.  
If yes, print **"Success"**; otherwise, print **"Failure"**.

## Example

| Input | Output   |
|------:|----------|
| 45    | Success  |
| 76    | Failure  |

## Approach
1. Read a two-digit integer.
2. Extract digit 0 (unit digit) and digit 1 (tens digit).
3. Compare digit 0 and digit 1 using `if/else`.
4. Print the result.

## Concepts Used
- Integer division
- Modulus operator
- Conditional statements
