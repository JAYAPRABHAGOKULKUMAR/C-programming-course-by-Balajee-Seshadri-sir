# C Level 4 – Problem 11

## Problem Statement
Write a program to get a number from the user and print the total number of digits.

## Example
Input: 123456 → Output: 6  
Input: 675 → Output: 3  

## Approach
1. Read the number.
2. Repeatedly divide by 10.
3. Count how many times division occurs.
4. Print the count.

## Concepts Used
- Digit counting
- Integer division
- goto loop
