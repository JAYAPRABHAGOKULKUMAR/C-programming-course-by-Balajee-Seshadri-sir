## Problem Statement
Write a loop program to print **odd numbers between 1 and 9**.

## Expected Output
1  
3  
5  
7  
9  

## Constraints
- Use only `int`
- No loops (`for`, `while`)
- Use `goto`
- No arrays or pointers

## Logic
- Start from 1
- Print the number
- Increment by 2
- Repeat until value exceeds 9
