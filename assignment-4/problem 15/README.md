# C Level 4 – Problem 15

## Problem Statement
Write a program to get a number from the user.  
If the last digit is even, print the same number.  
If the last digit is odd, subtract 1 from the last digit and print the number.

## Example
Input: 654324 → Output: 654324  
Input: 76895439 → Output: 66895439  
Input: 675 → Output: 575  

## Approach
1. Extract the last digit.
2. Check whether it is even or odd.
3. If odd, subtract 1 from last digit.
4. Replace the last digit and print the result.

## Concepts Used
- Modulus operator
- Conditional statements
- Integer arithmetic
