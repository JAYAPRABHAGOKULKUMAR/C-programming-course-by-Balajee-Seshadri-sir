# C Level 4 – Problem 12

## Problem Statement
Write a program to get a number from the user and print the sum of all digits.

## Example
Input: 123456 → Output: 21  
Input: 675 → Output: 18  

## Approach
1. Read the number.
2. Extract last digit using modulus.
3. Add digit to sum.
4. Remove digit using division.
5. Repeat until number becomes zero.

## Concepts Used
- Digit extraction
- Integer arithmetic
- goto-based looping
