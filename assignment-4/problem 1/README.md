## Problem Statement
Write a loop program to print numbers from **1 to 5** one by one.

## Expected Output
1  
2  
3  
4  
5  

## Constraints
- Use only `int` variables
- Do not use `for` or `while`
- Use only `if / else` and `goto`
- Do not use arrays, pointers, or strings

## Logic
- Initialize a variable with value 1
- Print the value
- Increment the variable
- Repeat using `goto` until the value reaches 5
