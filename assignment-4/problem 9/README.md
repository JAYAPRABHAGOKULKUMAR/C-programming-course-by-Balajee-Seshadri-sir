## Problem Statement
Write a loop program to print the two-digit even numbers whose sum of digits is 6.

## Expected Output
24  
42  
60  

## Approach
1. Start from 10 and go till 99.
2. Extract tens and units digits.
3. Check:
   - Number is even
   - Sum of digits equals 6
4. Print the number if conditions are satisfied.

## Concepts Used
- Integer arithmetic
- Digit extraction
- if / else
- goto-based looping
