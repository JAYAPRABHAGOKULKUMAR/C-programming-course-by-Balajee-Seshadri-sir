## Problem Statement
Write a loop program to print the **sum of numbers from 1 to 5**.

## Expected Output
15

## Constraints
- Use only `int`
- Loop using `goto`
- No `for` or `while`
- No arrays or pointers

## Logic

- Add each value to a sum variable
- Increment the counter
- Print the sum after loop ends
