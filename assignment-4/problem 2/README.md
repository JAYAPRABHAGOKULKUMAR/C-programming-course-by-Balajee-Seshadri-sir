## Problem Statement
Write a loop program to print numbers from **5 to 1** one by one.

## Expected Output
5  
4  
3  
2  
1  

## Constraints
- Only `int` variables
- No `for` or `while`
- Use `if / else` with `goto`
- No arrays or pointers

## Logic
- Start from 5
- Print the number
- Decrement the value
- Repeat until the value reaches 1
