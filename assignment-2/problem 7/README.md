# C Level 2 – Problem 7

## Problem Statement
Get a two-digit number from the user and check if **both digits are identical**.  
If yes, print `1`; otherwise, print `0`.

## Example

| Input | Output |
|------:|-------:|
| 55    | 1 |
| 56    | 0 |

## Approach
1. Extract ten’s digit and one’s digit.
2. Compare both digits using `==`.
3. Print the result.

## Concepts Used
- Integer division
- Modulo
- Relational operator (==)
