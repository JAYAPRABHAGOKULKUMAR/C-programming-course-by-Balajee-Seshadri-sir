## Problem Statement
Get a number from the user and check if the number is **less than or equal to 50**.  
If yes, print `1`; otherwise, print `0`.

## Example

| Input | Output |
|------:|-------:|
| 45    | 1 |
| 56789 | 0 |

## Approach
1. Read the number.
2. Compare using `<=`.
3. Print the result.

## Concepts Used
