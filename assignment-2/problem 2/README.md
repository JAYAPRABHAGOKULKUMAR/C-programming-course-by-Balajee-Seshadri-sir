## Problem Statement
Get a number from the user and check if the number is **not equal to 50**.  
If yes, print `1`; otherwise, print `0`.

## Example

| Input | Output |
|------:|-------:|
| 50    | 0 |
| 56789 | 1 |

## Approach
1. Read the number.
2. Compare the number with 50 using `!=`.
3. Print the result.

## Concepts Used
- Relational operator (!=)
