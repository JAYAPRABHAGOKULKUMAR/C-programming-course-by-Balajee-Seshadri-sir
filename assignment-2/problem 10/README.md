## Problem Statement
Get a two-digit number from the user and check if the digit 0 is greater than the digit 1.
If yes, print `1`; otherwise, print `0`.

## Example

| Input | Output |
|------:|-------:|
| 45    | 1 |
| 76    | 0 |

## Approach
1. Read the number.
2. Extract both digits.
3. Compare using the `>` operator.
4. Print the result.

## Concepts Used
- Integer arithmetic
- Relational operators
