## Problem Statement
Get a four-digit number from the user and check if digit 1 and digit 0 are the same.
If yes, print `1`; otherwise, print `0`.

## Example

| Input | Output |
|------:|-------:|
| 988   | 1 |
| 567   | 0 |

## Approach
1. Read the number.
2. Extract the last two digits.
3. Compare them using `==`.
4. Print the result.

## Concepts Used
- Modulus operator
- Equality operator
