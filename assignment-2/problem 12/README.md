## Problem Statement
Get a two-digit number from the user and check if digit 1 is greater than or equal to digit 0.
If yes, print `1`; otherwise, print `0`.

## Example

| Input | Output |
|------:|-------:|
| 66    | 1 |
| 67    | 0 |

## Approach
1. Read the number.
2. Extract digits.
3. Compare using `>=`.
4. Print the result.

## Concepts Used
- Integer arithmetic
- Relational operators
