## Problem Statement
Write a loop program to print numbers from 5 to 1 one by one.

## Explanation
The program prints numbers in reverse order starting from 5.

## Approach
1. Initialize `i` to 5.
2. Use a `while` loop until `i` becomes less than 1.
3. Print `i` and decrement it.
