## Problem Statement
Write a loop program to print the sum of numbers from 1 to 5.

## Explanation
The sum of numbers 1 + 2 + 3 + 4 + 5 is 15.

## Approach
1. Initialize `sum` to 0 and `i` to 1.
2. Add `i` to `sum` in each iteration.
3. Increment `i` until it reaches 5.
4. Print the final sum.
