## Problem Statement
Write a loop program to print numbers from 1 to 5 one by one.

## Explanation
The program prints numbers starting from 1 up to 5 in ascending order.

## Approach
1. Initialize a variable `i` to 1.
2. Use a `while` loop until `i` becomes greater than 5.
3. Print `i` and increment it.
