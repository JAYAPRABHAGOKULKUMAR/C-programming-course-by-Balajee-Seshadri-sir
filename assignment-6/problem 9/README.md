## Problem Statement
Write a loop program to print the sum of two-digit numbers whose one's digit is 5.

## Explanation
Numbers are: 15, 25, 35, 45, 55, 65, 75, 85, 95  
Their sum is 495.

## Approach
1. Start from 15.
2. Increment by 10 to maintain one's digit as 5.
3. Add each number to `sum`.
4. Print the final sum.
